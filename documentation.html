üìã Plugin Overview

The MMK Product Filter is a sophisticated WordPress plugin designed specifically for WooCommerce stores that need advanced product filtering capabilities. It creates 4-level dependent dropdown filters for Category, Make, Model, and Year attributes, making it perfect for automotive dealers, electronics stores, or any business with complex hierarchical product catalogs.

Perfect for: Automotive parts, Electronics, Machinery, Tools, Appliances, or any products with hierarchical categories and attributes.

Installation Methods

Method 1: WordPress Admin Upload

  1. Download the plugin ZIP file
  2. Go to WordPress Admin ‚Üí Plugins ‚Üí Add New
  3. Click "Upload Plugin" and select the ZIP file
  4. Click "Install Now" and then "Activate"

Method 2: FTP Upload

  1. Extract the plugin ZIP file
  2. Upload the mmk-product-filter folder to /wp-content/plugins/
  3. Go to WordPress Admin ‚Üí Plugins
  4. Find "MMK Product Filter" and click "Activate"

‚öôÔ∏è Configuration

Step 1: Prepare WooCommerce Categories & Attributes

Before configuring the plugin, ensure you have the necessary product categories and attributes set up in WooCommerce:

Product Categories

  1. Go to Products ‚Üí Categories
  2. Create your main product categories (e.g., "Electronics", "Automotive", "Tools")
  3. Ensure categories have products assigned to them

Product Attributes

  1. Go to WooCommerce ‚Üí Attributes
  2. Create attributes for your filtering needs:
    • Make/Brand: e.g., "Brand", "Manufacturer", "Make"
    • Model: e.g., "Model", "Series", "Type"
    • Year: e.g., "Year", "Model Year", "Production Year"
  3. Ensure attributes are set to "Used for variations" if needed
  4. Add terms to each attribute (e.g., Toyota, Honda for Make)

Step 2: Plugin Configuration

  1. Navigate to WordPress Admin ‚Üí Settings ‚Üí MMK Product Filter
  2. Configure the attribute mappings:
    • Make Attribute: Select your brand/manufacturer attribute
    • Model Attribute: Select your model/series attribute
    • Year Attribute: Select your year attribute
  3. Click "Save Changes"
Important: Make sure your products are assigned to categories and have these attributes assigned with appropriate values, or the filter won't display any options.

Step 3: Product Setup

For each product that should appear in filtered results:

  1. Edit the product in WooCommerce
  2. Assign the product to appropriate Categories
  3. Go to the "Attributes" tab
  4. Add the configured attributes and select appropriate values
  5. Save the product

üìñ Usage Instructions

Displaying the Filter

Use the shortcode to display the filter anywhere on your site:

[mmk_filter]

Common Placement Locations:

  • Shop Page: Add to shop page content or sidebar
  • Product Archive: Above product listings
  • Custom Pages: Landing pages or category pages
  • Widgets: In sidebar or footer widget areas

Filter Workflow

  1. Category Selection: User selects a category (e.g., Electronics)
  2. Make Population: Makes dropdown populates with relevant options for that category
  3. Make Selection: User selects a make (e.g., Samsung)
  4. Model Population: Models dropdown populates based on category + make
  5. Model Selection: User selects a model (e.g., Galaxy)
  6. Year Population: Years dropdown populates based on category + make + model
  7. Year Selection: User selects a year (e.g., 2023)
  8. Filter Action: Filter button becomes active
  9. Results: User clicks filter and sees filtered products

URL Structure

The plugin creates SEO-friendly URLs for filtered results:

https://yoursite.com/shop/?mmk_category=electronics&mmk_make=samsung&mmk_model=galaxy&mmk_year=2023
SEO Benefit: These URLs can be shared, bookmarked, and indexed by search engines.

üêõ Troubleshooting Guide

Common Issues & Solutions

‚ùå Filter Not Displaying

Symptoms: Shortcode shows but no dropdowns appear
Solutions:
  • Verify WooCommerce is active and working
  • Check if shortcode is correctly placed: [mmk_filter]
  • Ensure attributes are configured in plugin settings
  • Verify product categories exist and have products
  • Check browser console for JavaScript errors

‚ùå Empty Dropdown Options

Symptoms: Dropdowns show but have no options
Solutions:
  • Verify products are assigned to categories
  • Verify products have the configured attributes assigned
  • Check attribute mapping in Settings ‚Üí MMK Product Filter
  • Ensure attribute terms exist and are assigned to products
  • Verify products are published and visible

‚ùå AJAX Not Working

Symptoms: Dropdowns don't populate when selections are made
Solutions:
  • Check browser console for JavaScript errors
  • Ensure jQuery is loaded properly
  • Test with a default WordPress theme
  • Disable other plugins to check for conflicts

‚ùå No Products Filtered

Symptoms: Filter works but doesn't show filtered results
Solutions:
  • Ensure you're on the shop page or product archive
  • Verify product category and attribute assignments match filter selections
  • Check if other plugins are interfering with product queries
  • Test with WooCommerce default settings
  • Verify URL parameters are correct (mmk_category, mmk_make, mmk_model, mmk_year)

Debug Steps

  1. Enable WordPress Debug: Add to wp-config.php:
    define('WP_DEBUG', true); define('WP_DEBUG_LOG', true);
  2. Check Error Logs: Look in /wp-content/debug.log
  3. Browser Console: Check for JavaScript errors
  4. Network Tab: Monitor AJAX requests in browser dev tools
